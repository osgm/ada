# ADA 文档索引

## 文档分类

- `ADA-架构设计方案.md`  
  关注“为什么这样设计、模块边界是什么、演进路线如何规划”。
- `ADA-部署手册.md`  
  关注“如何安装、启动、运维、排障与发布验证”。
- `ADA-MCP-接入手册.md`  
  关注“如何通过 MCP 调用 ADA 能力、工具入参与示例”。
- `ADA-开发手册.md`  
  关注“开发流程、代码规范、测试门禁与本地联调”。
- `Playwright-ADA-兼容映射.md`  
  关注“Playwright 测试步骤如何映射到 ADA MCP 调用”。

## 使用建议

- 先读架构，再看部署，再根据角色看 MCP 或开发手册。
- 命令类内容以部署/开发手册为准，避免在架构文档重复维护。
- 接口类内容以 MCP 接入手册为准，避免在其他文档重复粘贴工具定义。

## 维护原则

- 架构文档不写细粒度命令步骤。
- 部署文档不展开底层设计推导。
- MCP 文档不重复通用部署说明。
- 开发文档不重复线上运维操作。
