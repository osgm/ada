# scripts 脚本清单

盘点 `scripts/` 下脚本的用途与 npm 引用。npm 发布流程见 [`README.md`](README.md)。

## 总览（8 个可执行脚本 + 2 个文档）

| 文件 | 建议 | npm script |
|------|------|------------|
| `build-mcp-npm.mjs` | 保留 | 经 `ada-mcp-server` → `build:npm` |
| `build-executable.mjs` | 保留 | `build:exe` |
| `generate-bundled-config.mjs` | 保留 | `generate:bundled-config` |
| `check-mcp-publish-versions.mjs` | 保留 | `mcp:check:versions` |
| `mcp-bundled-smoke.mjs` | 保留 | `test:mcp:bundled` |
| `e2e-smoke.mjs` | 保留 | `test:e2e:smoke*` |
| `verify-entrypoints.mjs` | 保留 | `test:entrypoints` |
| `sync-download-probe-vendor.mjs` | 按需 | 无（改 `packages/download-probe` 后手工执行） |

## 构建与发布

- **build-mcp-npm.mjs** — esbuild 打包 `@ada-mcp/mcp-server` 发布物。
- **generate-bundled-config.mjs** — 从 `config/default.yaml` 生成内嵌配置。
- **check-mcp-publish-versions.mjs** — 校验 mcp-server / launcher 版本号一致。
- **mcp-bundled-smoke.mjs** — 发布前冒烟：stdio 连 `dist/cli.cjs`，确认 21 工具与插件。
- **sync-download-probe-vendor.mjs** — 将 `packages/download-probe/dist` 同步到 launcher / mcp-server 内联脚本。

## 仓库级测试与打包

- **build-executable.mjs** — 打包四入口到 `release/`。
- **e2e-smoke.mjs** — 通过 ada-agent 跑真实任务集冒烟。
- **verify-entrypoints.mjs** — `build:exe` 后检查 `release/` 下可执行文件。
