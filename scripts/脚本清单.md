# scripts 脚本清单

目录结构：

```text
scripts/
├── build/       # 构建与工程
├── release/     # 发布门禁
├── test/        # 自动化测试
├── lib/         # 示例共用库（ada / fluent / popups 等）
└── examples/    # 业务示例脚本（薄，直接 import ../lib）
```

npm 发布流程见 [`README.md`](README.md)。

## build/ — ADA 工程脚本

| 文件 | npm script | 说明 |
|------|------------|------|
| `build-mcp-npm.mjs` | `build:npm -w @ada-mcp/mcp-server` | esbuild 打包 MCP 发布物 |
| `build-executable.mjs` | `build:exe` | 四入口 exe → `release/` |
| `generate-bundled-config.mjs` | `generate:bundled-config` | `config/default.yaml` → 内嵌 TS |
| `sync-download-probe-vendor.mjs` | （手工） | 同步 `download-probe` 到 launcher / mcp-server |

## release/ — 发布校验

| 文件 | npm script | 说明 |
|------|------------|------|
| `check-mcp-publish-versions.mjs` | `mcp:check:versions` | mcp-server / launcher 版本、Playwright pin、内联 mirror 一致 |
| `sync-download-probe-vendor.mjs` | `sync:download-probe` | 改 download-probe 后同步 launcher / mcp-server 内联脚本 |
| `sync-log-locale-vendor.mjs` | `sync:log-locale` | 改 log-locale.ts 后同步 launcher / mcp-server |
| `sync-scripts-lib-from-driver-rpc.mjs` | `sync:scripts-lib` | 改 driver-rpc swipe/fillSearch 后同步 scripts/lib |
| `mcp-bundled-smoke.mjs` | `test:mcp:bundled` | 发布包 stdio 冒烟 |

## test/ — 仓库测试

| 文件 | npm script | 说明 |
|------|------------|------|
| `e2e-smoke.mjs` | `test:e2e:smoke*` | ada-agent 任务集冒烟 |
| `verify-entrypoints.mjs` | `test:entrypoints` | `build:exe` 后检查 `release/` |
| `test-hidden-spawn.mjs` | — | Windows `windowsHide` 排障 |

## lib/ — 示例共用库

| 文件 | 说明 |
|------|------|
| **`ada-client.mjs`** | JS 统一入口 |
| **`ada_client.py`** | Python 统一入口 |
| `ada.mjs`、`ada-fluent.mjs`、`popups.mjs` 等 | 内部实现模块 |
| `run-python-example.mjs` | 运行 Python 示例（设置 `PYTHONPATH`） |

Python 统一入口：`lib/ada_client.py`（`npm run test:jd-*:py` 或 `PYTHONPATH=scripts/lib`）。

文档：[`lib/README.md`](lib/README.md)。

## examples/ — 京东 E2E（按平台分目录）

| 路径 | npm script | 说明 |
|------|------------|------|
| `nodejs/web/jd-e2e.mjs` | `test:jd-web` | Web 4 场景 |
| `nodejs/android/jd-e2e.mjs` | `test:jd-android` | Android 10 步 |
| `nodejs/harmony/jd-e2e.mjs` | `test:jd-harmony` | 鸿蒙 10 步 |
| `python/{web,android,harmony}/jd_e2e.py` | `test:jd-*:py` | Python 对照 |

文档：[`examples/README.md`](examples/README.md)。
